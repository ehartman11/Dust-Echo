AI Music Library

Dust, neon, prayer, and backroad memory in one roots-built collection.

Dust & Echo is a ten-song AI-assisted music library shaped around American folk, blues-rock, and Red Dirt country. Choose a track from the library to open its cover art, full lyric sheet or arrangement notes, short description, and playable audio in the listening panel.

Project Notes

          
A curated AI music library with a roots-country center.

          

            The project blends narrative songwriting, blues-rock energy, and rural imagery into a single
            listening experience. Instead of showing every song as an equal-size block, the site now gives
            each track a focused detail view that makes the lyrics and concept easier to engage with.

Publish Path

          
Final steps from local library to public GitHub Pages site.

        


        

          01
          
Review the sequence

          
Click through the library and make sure the current alternate exports are the ones you want representing each song.

        

        

          02
          
Push to GitHub

          
Commit the site, media, and docs to your target repository so the full static project lives in one place.

        

        

          03
          
Deploy from root

          
Publish the branch root with GitHub Pages so `index.html`, `styles.css`, `script.js`, and `assets/` all resolve together.

        

        

          04
          
Share the link

          
Once the Pages build is live, the site becomes a single shareable portfolio page for the full Dust & Echo collection.
